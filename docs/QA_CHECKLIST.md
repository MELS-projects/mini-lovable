# QA Checklist

Before accepting a new version:

- App preview loads without error
- Mini-Lovable UI still works
- DeepSeek API key field exists
- Generate / Update works
- Create Roadmap works
- Build / Improve Step works
- Run App Review works
- Export buttons exist
- Contact form validation rules are preserved
- User-facing Mini-Lovable UI stays in English
- No backend, auth, database, payments, or deploy features added
- Ground Zero quality is not reduced
- Existing saved version can be restored if something breaks
- Changes are small and focused
- No unnecessary rewrite of unrelated sections

# Ground Zero

Current accepted baseline:
v52.16 contact validator order fix

This version is the minimum acceptable baseline for the Mini-Lovable tool.

Future updates must not:

- Reduce UI quality
- Remove roadmap tools
- Remove review tools
- Break generated app preview
- Break export tools
- Weaken contact form validation
- Add unnecessary technical complexity
- Add auth/login
- Add database
- Add payments
- Add backend workflows
- Add deployment features yet

Definition of acceptable progress:
A new version is only better if it preserves the current tool and improves one focused part without breaking existing functionality.
